## 🔐 Key System

С версии v0.1.0 добавлена система входа по ключам. После запуска скрипта появляется окно авторизации, где необходимо ввести валидный ключ формата `LSMM2-XXXXXX-X-XX`. Без ключа скрипт не загрузится. Всего доступно **20 уникальных ключей**.

| Формат ключа | Пример |
|-------------|--------|
| `LSMM2-XXXXXX-X-XX` | `LSMM2-A1B2C3-D-EF` |

> Для получения ключа обращайтесь к боту: [@LessScriptArsenal-Bot](https://t.me/LessScriptArsenal-Bot)

## 🔥 Возможности

### 👁️ ESP (Визуальные)
| Функция | Описание |
|---------|----------|
| **Sheriff ESP** | Подсветка шерифа синим цветом через стены |
| **Murderer ESP** | Подсветка убийцы красным цветом через стены |
| **Reveal Roles** | Автоматическое определение и обновление ролей |
| **Player ESP** | Имена, HP и расстояние до всех игроков |
| **Gun ESP** | Подсветка оружия жёлтым цветом |
| **Coin ESP** | Подсветка монет золотым цветом |
| **Throw ESP** | Подсветка летящих ножей убийцы |
| **Map ESP** | Мини-карта с позициями всех игроков (Sheriff синий, Murderer красный) |

### 🎯 Combat (Боевые)
| Функция | Описание |
|---------|----------|

| **Silent Aim** | Скрытое наведение без визуального движения камеры |
| **Knife Aimbot** | Наведение на убийцу для удара ножом |
| **Auto Stab** | Автоматический удар ножом при наведении на убийцу |
| **Auto Shoot** | Автоматическая стрельба по ближайшей цели |
| **FOV Circle** | Визуальный круг аимбота |

## 📱 Установка

### Требования
- Roblox Mobile (iOS/Android)
- Любой мобильный эксплоит (Arceus X, Hydrogen, Fluxus Mobile, CodeX, Delta и др.)
- Валидный ключ доступа (формат: LSMM2-XXXXXX-X-XX)

1. Скачай и установи мобильный эксплоит
